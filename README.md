# Daily Focus Timer

A minimalist Pomodoro web app for deep work, built using frontend-only logic.

## Features

* 25/5 Pomodoro mode
* Start, pause, reset controls
* Session counter
* Dark mode toggle
* Sound alerts
* No backend, no database

## Stack

* TypeScript, NextJS, Tailwind CSS

## Use Case

Designed for distraction-free focus and simple productivity tracking.
