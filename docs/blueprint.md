# **App Name**: FocusFlow

## Core Features:

- Timer Control: Start, pause, and reset the Pomodoro timer.
- 25/5 Mode: Switch between 25-minute work sessions and 5-minute break sessions.
- Session Counter: Track the number of Pomodoro sessions completed.
- Dark Mode Toggle: Enable or disable dark mode for better viewing in different lighting conditions.
- Sound Alerts: Play sound alerts to indicate the start and end of work and break sessions.

## Style Guidelines:

- Primary color: Deep navy blue (#243A73) to promote a sense of focus and calm.
- Background color: Very dark gray (#121212) for a minimalist dark mode aesthetic.
- Accent color: Soft lavender (#B1A2CA) to highlight key UI elements.
- Body and headline font: 'Inter', a grotesque sans-serif, provides a modern, machined look for readability.
- Simple, outlined icons for start, pause, reset, and settings.
- Clean, centered layout for optimal focus on the timer.
- Subtle, non-distracting animations for transitions and alerts.
